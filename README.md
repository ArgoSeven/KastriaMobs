# KastriaMobs

**KastriaMobs** is a Fabric mod that adds 9 unique hostile mobs to Minecraft, each with custom 3D models (powered by Geckolib), distinctive attacks, and configurable stats.

## Mobs

| Mob | Description |
|-----|-------------|
| **Bastion** | A heavily armored tank with high durability. Melee attacks apply the Wither effect. |
| **Blindwrath** | Uses concentrated sonic beam attacks to damage and knock back targets from a distance. |
| **Hollowseer** | A spellcaster that summons Evoker fangs in a linear pattern and fires cursed bullets that inflict Poison. |
| **PlagueBrute** | A massive brute that releases devastating sonic boom explosions. Melee attacks apply Slowness. |
| **Reaver** | Summons Evoker fangs in expanding circular patterns around targets. Applies Blindness on melee hits. |
| **RedBloodMage** | Fires a blood-red energy beam that pulls enemies toward the caster while dealing damage. |
| **Stalker** | A swift melee attacker with high armor. Inflicts extended Wither effect on hit. |
| **Bard** | A resilient mob that reflects projectile attacks with sonic notes. Fires a powerful sonic beam when provoked. |
| **Tobias** | An alchemist that intelligently selects potions based on target distance and health - uses Slowness at range, Poison on healthy targets, and Harming up close. |

## Configuration
All mob attributes are editable including:
- Health, movement speed, attack damage
- Armor and armor toughness
- Attack cooldowns, ranges, and damage values
- Status effect type, duration, and amplifier for melee attacks

## Commands

- `/kastria reload` - Reloads the mod configuration without restarting

## Dependencies

- Fabric API
- Geckolib 3.x
